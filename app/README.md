# App

Partie **app** du projet Webtech 2022

## Prérequis

Next.js

Assurez-vous d'être dans le dossier */app* :

```bash
cd ./app
```

## Essayer le code

Commencez par initialiser npm :

```bash
npm istall 
npm init -y
```

Puis :

```bash
npm run dev
```

Accedez à la page d'accueil via : <http://localhost:3000/>
